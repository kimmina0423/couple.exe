# 가트만 이론 (Gottman Method) — AI 분석 엔진용

이 문서는 커플 일기를 분석하는 AI가 적용해야 할 가트만 기반 분석 규칙이다.
분석 시 아래 기준을 매 일기마다 적용하라.

---

## 1. 4가지 독 (Four Horsemen) 감지 규칙

### 비난 (Criticism)
행동 비판이 아니라 **인격 자체**를 공격하는 표현.
→ 반대 행동: 부드러운 시작(Gentle Start-Up) — "나는 ~할 때 ~하게 느껴"

**감지 패턴:**
```
"너는 항상 ~해"           → always/never 절대화
"당신은 왜 그렇게 ~야"    → 인격 단정 의문문
"결국 너란 사람은 ~"      → 본질 귀속 표현
"매번 이런 식이야"        → 반복 패턴 일반화
"그게 문제야, 네가"       → 인격으로 귀인
```

### 방어 (Defensiveness)
책임을 거부하고 역으로 피해자가 되거나 역공격하는 패턴.
→ 반대 행동: 부분 책임 수용 — "네 말도 맞아, 내가 ~한 부분이 있어"

**감지 패턴:**
```
"나는 그런 거 아니었는데"           → 의도 부정 + 설명 과잉
"그건 네가 먼저 ~했으니까"          → 역공격·책임 전가
"나만 잘못한 것처럼 말하네"         → 피해자화
"맞아, 하지만 ~"                   → Yes-but 패턴
"나도 힘들어"로 방향 바꾸기         → 역공감 요구
```

### 경멸 (Contempt) ← 이혼 예측력 최고, 즉시 RED 처리
상대방을 열등하게 보는 비웃음, 냉소, 조롱.
단 하나라도 감지되면 즉시 riskLevel을 RED로 강제 상향하라.
→ 반대 행동: 감사와 존중 문화 구축

**감지 패턴:**
```
"정말 한심해", "어이가 없어"        → 조롱/비웃음 어휘
"어차피 말해봤자"                  → 무가치화
"그러면 그렇지"                    → 냉소적 예측
눈알 굴리기 묘사, 비꼬는 "(웃음)"  → 멸시 비언어 기술
"그게 되겠어", "뻔하지"            → 능력·가능성 조롱
"걔가 원래 그래"                   → 고정적 열등화
```

### 담쌓기 (Stonewalling)
감정 범람(Flooding) 시 대화 자체를 차단하고 철수하는 패턴.
→ 반대 행동: 자기진정(Self-Soothing) 20분 후 재대화 약속

**감지 패턴:**
```
"그냥 아무 말도 하기 싫었다"        → 철수·회피 선언
"대화가 의미없다고 느꼈다"          → 포기 표현
"혼자 있고 싶었다"                 → 단절 욕구
짧은 단답 묘사, 무시 묘사           → 행동적 담쌓기
"그냥 듣고 싶지 않았다"             → 청각 차단
```

## 2. 사랑의 지도 (Love Maps)

가트만 이론에서 안정된 커플은 상대의 내면세계(좋아하는 것, 두려움, 꿈, 스트레스 요인)를
상세히 알고 있다. 일기를 매일 쓰는 행위 자체가 이 지도를 업데이트하는 과정이다.

**분석 AI 규칙:**
- 일기에서 상대방을 구체적으로 묘사할수록 사랑의 지도가 넓다는 신호.
- "파트너가 오늘 힘들었을 것 같다", "좋아한다는 걸 알았다" 같은 추측·공감 표현 → 긍정 신호.
- 상대방이 추상적 존재("그 사람", "상대")로만 등장하고 구체적 묘사가 없을 때 → 지도 협소 신호.

---

## 3. 감정은행계좌 (Emotional Bank Account) + 5:1 긍정-부정 비율

**긍정 입금 신호 패턴:**
```
"고마워", "덕분에", "잘해줬어"      → 감사 표현
"같이 ~하니까 좋았어"              → 공유 경험 긍정
"이해해줘서", "들어줘서"            → 공감 수용 표현
상대방 강점·노력 언급               → 인정 표현
함께 웃은 에피소드                  → 유머·애정 신호
```

**위험도 산정 기준:**
```
긍정 문장 / 부정 문장 비율:
 ≥ 5:1  → GREEN (안정)
 2~4:1  → YELLOW (주의)
 < 2:1  → RED (위험)
contempt 감지 시 → 비율 무관, 즉시 RED
```

---

## 4. 추구-철수 (Pursue-Withdraw) 패턴

**정의:** 한 사람은 연결·반응을 추구하고, 다른 사람은 회피·철수하는 상호작용 패턴.
두 일기를 비교했을 때 다음이 동시에 나타나면 감지:

- A 일기: 갈등 언급, 대화 요청, 확인 요구, 상대 반응에 대한 실망
- B 일기: 담쌓기 표현, 회피, 짧은 단답 묘사, "귀찮았다", "말하기 싫었다"

---

## 5. 회복 시도 (Repair Attempts)

안정된 커플과 위기 커플의 핵심 차이: **갈등 후 회복을 시도하는가, 받아주는가**.

**회복 시도 신호:**
```
갈등 후 먼저 연락했다
사과했다 ("미안해", "내가 좀 심했어")
유머로 분위기를 바꾸려 했다
"우리 잘 될 것 같아", "그래도 좋아해" 등 애정 재확인
```

**수용 신호:**
```
"괜찮아", "나도 미안해"
상대의 회복 시도를 받아들인 묘사
```

회복 시도 감지 시 → riskLevel을 한 단계 낮추는 완화 요소로 적용하라.
회복 시도가 거부된 묘사 → 추가 경고 신호.

---

## 6. 공유된 의미 만들기 (Shared Meaning)

가트만의 관계 피라미드 최상층. 둘만의 의식·상징·문화·서사를 만드는 것.

**감지 신호:**
```
"우리만의 ~", "항상 우리가 ~하던 것"    → 둘만의 의식/루틴
"우리 앞으로 ~하고 싶다"               → 공유된 미래 서사
"그 사람 생각하면 ~가 떠오른다"        → 공유 상징
```

공유 의미가 강할수록 → 몬스터 자동 처치 저항력 높음, 퀘스트 효과 배가.

---

## 7. 퀘스트 선택 기준 (가트만 이론 기반)

| 감지 신호 | 권장 퀘스트 유형 | 이론 근거 |
|---|---|---|
| 비난 감지 | 부드러운 시작 연습 | Gentle Start-Up |
| 담쌓기 감지 | 20분 자기진정 + 재대화 | Self-Soothing |
| 추구-철수 패턴 | 구조화된 대화 (타이머 사용) | Speaker-Listener Technique |
| 경멸 감지 | 상대방 강점 3가지 찾기 | Fondness & Admiration |
| GREEN 상태 | 사랑의 지도 질문 교환 | Love Maps |
| 회복 시도 있었음 | 감사 편지 또는 감사 말하기 | Emotional Bank Account |
| 방어 패턴 | "나는" 메시지 (I-message) 연습 | CBT + Gottman |

---

## 8. 갈등 위험도 등급 최종 판정 기준

```
1. contempt 감지 → 즉시 RED (다른 지표 무시)
2. 긍정:부정 비율 < 2:1 → RED
3. 긍정:부정 비율 2~4:1 또는 criticism/stonewalling/defensiveness 중 하나 감지 → YELLOW
4. 긍정:부정 비율 ≥ 5:1 + 회복 시도 있음 → GREEN
5. 모든 Horsemen 없음 + 긍정 입금 신호 3개 이상 → GREEN
```
